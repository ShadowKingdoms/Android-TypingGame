# Android-TypingGame 소개
안드로이드 스튜디오와 MySQL을 기반으로 제작한 모바일용 타자연습 어플리케이션입니다. 게임 시작 전 다양한 난이도와 플레이 시간 등의 설정이 가능하며 단어 종류는 한/영 단어를 지원합니다. 게임 종료 시 점수를 DB에 기록할 수 있어서 언제든지 자신의 기록을 열람할 수 있습니다.
